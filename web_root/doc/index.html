ZIO Quartz H2 Example

This is a demonstration of serving static files with ZIO Quartz H2 HTTP/2 server.

Available Routes

  • / Returns a simple OK response
  • /hello Returns a greeting message. Add ?name=YourName to personalize it
  • /doc/* Serves static files from the web_root directory

Server Features

  • HTTP/2 protocol support with TLS
  • Efficient static file serving with automatic content type detection
  • Functional route definition using ZIO
  • Streaming response support
  • Middleware support for request/response transformation
